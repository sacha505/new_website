Compile a Document

Single .tex file:

pdflatex document.tex

Alternatively, you can use something online like Overleaf.

Include Packages

All packages should to be included before the \begindocument command

\documentclass[12pt, letterpaper]{report}
\usepackage
\title{Your Title}
\begin{document}
\end{document}

Create a Contents Page

This is fairly simple:

\tableofcontents

Add a Bibliography

The most organized way to do this is to use BibTex. You can manage your bibliography manually but it's a pain.

You'll need to make a .bib file which will contain the each reference in your paper. You need to make sure these are accurate.
Each individual reference will look something like this:

@online{website, author = "Jane Doe",
title = "My Book",
url = "https://www.wikipedia.org/",
addendum = "(accessed: 01.01.2010)",
keywords = "latex, books, guides" }

Adding and Positioning Images

To add an image, it needs to be in a known directory to the interpreter. One common mistake is not to have included the right file path.

The easiest way to position images is by using float, represented here with the [H]:

\begin{figure}[H]
\centering
\includegraphics[width=0.5\linewidth]{pathtoyour/file/image.png}
\caption{your caption here}
\end{figure}

Float tells LaTeX to put your image in the same place it appears in the code. You can also see here there is a width modifier which you can use to resize your image.

Adding Code Snippets

The \lstset environment lets us write example code

\usepackage{listings}
\begin{lstlisting}[language=C]
%your code here
\end{lstlisting}

Lists and Sublists

Plenty of ways to make lists in LaTeX, and you can nest lists inside of lists.

\begin {itemize}
\begin {itemize}
\begin {itemize}
\end {itemize}
\end {itemize}
\end {itemize}

Escape Characters

Some characters are considered 'special' in LaTeX so won't render if you type them without an escape character, which is '\'.

\%
